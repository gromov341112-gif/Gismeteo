# Gismeteo Precipitation

Текущая версия: `2.3`

## Возможности

- Точный поиск прогноза по указанному городу без подмены на похожие локации.
- Сбор прогноза Gismeteo на 10 дней для каждого города из списка.
- Выгрузка ежедневных осадков в миллиметрах.
- Сбор ветра и порывов ветра в м/с по основному городу.
- Сохранение описания погоды в том виде, как оно указано на Gismeteo.
- Формирование Excel-отчета с основным листом `Отчет`.
- Создание листа `Осадки` с городами и датами, где осадки превышают 5 мм.
- Создание листа `Осадки-Порывы` с городами и датами, где есть осадки или порывы ветра.
- Поддержка фиксированного списка городов на листе `Осадки-Порывы`: если список пустой, показываются все города из отчета.
- Выбор даты на листе `Осадки-Порывы` через отдельное поле без нестабильной привязки к Excel-фильтру.
- Наглядные диаграммы осадков, ветра и порывов по каждому городу.
- Кликабельные ссылки на страницы прогноза.
- Автофильтр по текущему дню на листе `Осадки`.
