# Gismeteo Precipitation

Текущая версия: `2.3`

## Возможности

* Точный поиск городов без подмены на похожие локации.
* Сбор 10-дневного прогноза с Gismeteo.
* Выгрузка осадков (мм), ветра и порывов ветра (м/с).
* Сохранение оригинального описания погодных условий.
* Автоматическое формирование Excel-отчёта.
* Лист **«Осадки»** с днями, где ожидается более 5 мм осадков.
* Лист **«Осадки–Порывы»** с фильтрацией по выбранной дате и поддержкой фиксированного списка городов.
* Наглядные диаграммы по осадкам, ветру и порывам.
* Кликабельные ссылки на страницы прогноза Gismeteo.
* Автофильтр по текущему дню.
